# 

## Music API
### Description

A music API application that matches the name of songs to the artist that they belong to. Songs are also matched to specific albums from a particular artist.

### Installation

First run:
```js
npm init
```

Then run:
```js
npm start
```

### Built With

* Javascript
* Mongoose
* Node.js
